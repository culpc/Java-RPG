-You will need to have JavaFX and jFugue installed for the program to work.
-Compile all .java files and run TestEnginev2.java

-The program will only respond to the correct inputs (forward, search, slash, stab etc.) which are offered in the prompts.
-To close the program, type quit in the prompt or end the run in jGrasp.
-Clicking the X at the end of the game will not end the music, so in this case you must end the run.
